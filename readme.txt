== Description ==

Clarion is a classic and elegant WordPress blog theme built for writers and storytellers. It features beautiful serif typography (Playfair Display for headings, Merriweather for body text), warm gold accents, a clean sidebar layout, and a timeless reading experience.

Key features include:
- Classic & elegant design with serif typography
- Responsive, mobile-first layout
- Customizer options: accent color, background color, text color, sidebar position, posts per row
- Right sidebar, left sidebar, or full-width layout
- Author bio box on single posts
- Related posts section (no plugins needed)
- Estimated reading time display
- Breadcrumb navigation (no plugins needed)
- Footer widget areas (3 columns)
- Social links menu
- Gutenberg block editor support with matching editor styles
- Dark mode support via prefers-color-scheme
- Full accessibility: skip links, ARIA roles, keyboard navigation, focus styles
- Translation ready


== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

Or install via FTP:
1. Upload the `clarion` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.


== Frequently Asked Questions ==

= Does this theme support any page builders? =

Clarion is designed to work beautifully with the WordPress block editor (Gutenberg). Other page builders may work but are not officially tested.

= How do I set up the logo? =

Go to Appearance > Customize > Site Identity and upload your logo image.

= How do I set up the navigation menus? =

Go to Appearance > Menus to create and assign menus to the Primary Menu, Footer Menu, and Social Links Menu locations.

= How do I use the Social Links menu? =

Create a menu and add Custom Links for each social network URL. The theme will display them as icon links in the footer.

= How do I add widgets to the sidebar and footer? =

Go to Appearance > Widgets and add widgets to the Sidebar, Footer Column 1, Footer Column 2, and Footer Column 3 areas.

= How do I enable/disable the author bio box? =

Go to Appearance > Customize > Theme Options > Blog Options and toggle "Show Author Bio on Single Posts".

= Can I change the accent color? =

Yes. Go to Appearance > Customize > Theme Options > Colors and pick your preferred accent color.

= Where do I find font files? =

Playfair Display and Merriweather font files (WOFF2 format) need to be downloaded from Google Fonts and placed in:
- assets/fonts/playfair-display/
- assets/fonts/merriweather/

See the fonts.css file for the expected file names.
